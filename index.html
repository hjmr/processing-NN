• Clicking nodes toggle ON(red sign is also on) and OFF of input and teaching signals.
  • Clicking rectangle "M" button registers the current set of input and teaching signals to the training queue.
  • Clicking rectangle "R" button recalls a set of input and teaching signals from the queue.
  • Clicking circle "T" button starts training by using the current training queue.
  • Clicking circle "R" button starts running.
